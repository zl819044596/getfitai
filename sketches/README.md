# GetFitAI 核心功能页草图对比

## 三个设计方向

| 维度 | 001-Stepper Wizard | 002-Card Tabs | 003-Hero Focused |
|:---|:---|:---|:---|
| **核心思路** | 分步引导，降低决策压力 | 标签切换，信息层级清晰 | 深色沉浸，视觉冲击强 |
| **布局** | 左图右表，顶部进度条 | 左图右表，顶部标签 | 全屏深色，左右分栏 |
| **表单交互** | 一步一题，2×2 网格选项 | 标签切换，列表式选项 | 直接展示，玻璃拟态按钮 |
| **视觉风格** | 白底清爽，圆角柔和 | 白底现代，卡片突出 | 深色渐变，玻璃质感 |
| **信息密度** | 中等，聚焦当前步骤 | 较高，所有步骤可见 | 低，只展示核心入口 |
| **社交证明** | 底部头像+数字 | 顶部小字 | 顶部头像+数字 |
| **左侧图片** | 单图+底部数据卡片 | 单图+两侧浮动数据卡 | 单图+浮动UI卡片+进度环 |

---

## 各方案优劣

### 001-Stepper Wizard ⭐ 推荐
- **强**：分步降低认知负荷，用户不会看到一堆问题就退缩；进度条有成就感
- **弱**：需要多步操作， impatient 用户可能流失
- **适合**：首次使用、对健身不太了解的用户

### 002-Card Tabs
- **强**：所有步骤一目了然，power user 可以快速切换；列表式选项信息更全
- **弱**：一次性展示太多，可能让新手感到复杂
- **适合**：熟悉健身、知道自己要什么的用户

### 003-Hero Focused
- **强**：视觉冲击力强，品牌感强；深色模式差异化明显
- **弱**：表单功能被弱化，转化率可能受影响；开发成本高
- **适合**：品牌展示页、Landing Page，而非核心功能页

---

## 我的建议

**主方案：001-Stepper Wizard**
- 健身计划生成器面向的多数是普通用户，不是专业教练
- 分步引导能显著提高完成率（行业数据：分步表单比单页长表单转化率高 20-30%）
- 可以加入「快速模式」按钮，老用户一键跳过

**可选融合**
- 001 的进度条 + 002 的列表式选项卡片
- 001 的布局 + 003 的深色头部（仅顶部）
