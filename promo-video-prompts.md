# GetFitAI 宣传片 AI 生成提示词

## 项目信息
- **产品**: GetFitAI - AI 健身计划生成器
- **核心功能**: 回答5个问题，AI生成个性化训练计划
- **目标用户**: 健身新手到进阶者
- **视觉风格**: 深色主题(slate-950) + 橙色强调(#f97316)
- **品牌调性**: 专业、科技感、激励、现代

---

## 方案一：产品界面演示风（推荐）

### 镜头1 - 开场 (0-3秒)
```
Dark navy background with subtle orange gradient glow. 
A sleek smartphone screen emerges from darkness, displaying 
"GetFitAI" logo with orange dumbbell icon. 
Cinematic lighting, shallow depth of field, 4K quality.
Text overlay: "Your AI Fitness Coach"
```

### 镜头2 - 问题选择 (3-8秒)
```
Close-up of mobile app interface on dark background. 
Clean UI showing 4 goal options: "Build Muscle", "Lose Fat", 
"Get Stronger", "Boost Endurance" with icons. 
A finger taps "Build Muscle" - option highlights in orange. 
Smooth micro-interaction animation, glassmorphism cards.
```

### 镜头3 - 智能生成 (8-12秒)
```
Abstract visualization of AI processing. 
Dark background with flowing orange particle streams 
forming a human muscle anatomy silhouette. 
Digital nodes connecting, data flowing. 
Text: "AI analyzing your profile..." 
Futuristic, tech aesthetic, 60fps smooth motion.
```

### 镜头4 - 计划展示 (12-18秒)
```
Mobile screen displaying generated workout plan. 
Dark UI with orange accents. Shows: "Upper Body Power" 
with exercise list, sets/reps, rest timers. 
Card flips and slides in with smooth animation. 
Professional fitness app interface design.
```

### 镜头5 - 健身场景混剪 (18-25秒)
```
High-energy montage of diverse people training:
- Woman doing deadlifts in modern gym, dramatic lighting
- Man performing pull-ups, silhouette against window
- Person using dumbbells at home, focused expression
- Athlete doing yoga stretch, calm and controlled
All shots: cinematic color grading, dark moody tones 
with orange accent lighting, slow motion emphasis.
```

### 镜头6 - 数据/成果展示 (25-30秒)
```
Split screen: Left shows person exercising, 
Right shows GetFitAI dashboard with progress charts. 
Metrics animate up: "12,847 plans generated", 
"4.9 user rating", "30s generation time". 
Dark theme UI, orange data visualizations, 
glassmorphism cards floating in space.
```

### 镜头7 - 结尾 CTA (30-33秒)
```
Logo animation: Orange dumbbell icon morphs into 
"GetFitAI" wordmark on dark background. 
Tagline fades in: "Stop Guessing. Start Growing." 
CTA button: "Get My Free Plan →" 
Subtle orange glow pulse, cinematic fade out.
```
